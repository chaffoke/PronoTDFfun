🔐 Code administrateur
Chaque personne choisit son profil au premier lancement (via le bouton 👤 en haut) : son propre nom (ne peut modifier que ses pronostics des étapes à venir), invité (lecture seule), ou administrateur (accès complet — points, bonus, équipes, réglages). Ce code protège l'accès au profil Administrateur. Laisse vide pour que n'importe qui puisse devenir administrateur sans code.
